## 鼠标

### 1. 点击

- 分类：`鼠标`
- 实现状态：`已实现`
- 语法：`点击 左键|右键 单点|双击|多击|长按 (x,y) [次数或秒数]`
- 说明：在指定坐标执行鼠标点击。
- 参数说明：
  - 鼠标按键参数支持 `左键` 或 `右键`。
  - 点击模式参数支持 `单点`、`双击`、`多击`、`长按`。
  - `(x,y)` 表示单个屏幕坐标。
  - `秒数` 支持整数或小数。
  - `次数` 使用整数。
  - 方括号中的内容表示可选参数。
- 返回/变量回写：这条命令主要执行动作本身，不强制产生返回值。
- 最小示例：
```os
点击 左键 单点 (100,100) 2
```
